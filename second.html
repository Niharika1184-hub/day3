The Standard CSS Box Model

In a standard css box model.the height and width attributes is specified.this determines the size of the content area.if the padding,border,and margin properties are also specified,then they will be added to the width and height to calculate the total sizeof the element

the text makes up the content layer of the box.since i've added 30px padding,10px border,and 10px margin,the total width of the box is 500.you get that by adding the content width,padding-left and padding-right,border-left and border-right,and margin-left and margin-right(400+30+30+10+10+10+10).the total height of the box is 250.you get that by adding the content-height,padding-top and padding-bottom,border-top and border-bottom, and margin-top and margin-bottom(150+30+30+10+10+10+10).
